# Quick Note Application

## Overview
This project is a simple single-page note-taking application built using Python Flask and SQLite. Users can write notes, save them to a database, and view saved notes on the same page.

## Features
- Add and save notes
- Store notes in SQLite database
- Display saved notes instantly
- Simple request-response workflow

## Technologies Used
- Python
- Flask
- SQLite
- HTML
